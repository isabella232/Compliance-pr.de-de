## <a name="microsoft-and-csa-star-certification"></a><span data-ttu-id="3a042-112">Microsoft und die CSA STAR-Zertifizierung</span><span class="sxs-lookup"><span data-stu-id="3a042-112">Microsoft and CSA STAR certification</span></span>

<span data-ttu-id="3a042-113">Microsoft Azure, Microsoft Intune und Microsoft Power BI haben eine STAR-Zertifizierung erhalten, die eine strenge und unabhängige Bewertung der Sicherheitslage eines Cloudanbieters durch Dritte erfordert.</span><span class="sxs-lookup"><span data-stu-id="3a042-113">Microsoft Azure, Microsoft Intune, and Microsoft Power BI have obtained STAR Certification, which involves a rigorous independent third-party assessment of a cloud provider’s security posture.</span></span> <span data-ttu-id="3a042-114">Für diese STAR-Zertifizierung ist das Erreichen der ISO/IEC 27001-Zertifizierung und die Erfüllung der in der CCM angegebenen Kriterien erforderlich.</span><span class="sxs-lookup"><span data-stu-id="3a042-114">This STAR certification is based on achieving ISO/IEC 27001 certification and meeting criteria specified in the CCM.</span></span> <span data-ttu-id="3a042-115">Mit der Zertifizierung wird nachgewiesen, dass ein Cloud Service-Anbieter mit den entsprechenden Anforderungen nach ISO/IEC 27001 konform ist, wichtige Probleme der Cloudsicherheit, wie in der CCM ausgeführt, berücksichtigt hat und im Rahmen des STAR Capability Maturity Model im Hinblick auf die Verwaltung der Aktivitäten in CCM-Kontrollbereichen bewertet wurde.</span><span class="sxs-lookup"><span data-stu-id="3a042-115">It demonstrates that a cloud service provider conforms to the applicable requirements of ISO/IEC 27001, has addressed issues critical to cloud security as outlined in the CCM, and has been assessed against the STAR Capability Maturity Model for the management of activities in CCM control areas.</span></span>  
  
<span data-ttu-id="3a042-116">Während der Bewertung weist ein akkreditierter CSA-Zertifizierungsprüfer jedem der 16 CCM-Kontrollbereiche eine Maturity Capability-Punktzahl zu.</span><span class="sxs-lookup"><span data-stu-id="3a042-116">During the assessment, an accredited CSA certification auditor assigns a Maturity Capability score to each of the 16 CCM control areas.</span></span> <span data-ttu-id="3a042-117">Anhand der durchschnittlichen Punktzahl wird die Gesamtreife ermittelt und der entsprechende Bronze, Silver oder Gold Award verliehen.</span><span class="sxs-lookup"><span data-stu-id="3a042-117">The average score is then used to assign the overall level of maturity and the corresponding Bronze, Silver, or Gold award.</span></span> <span data-ttu-id="3a042-118">Azure, Intune, Power BI und Microsoft Cloud App Security erhielten die Cloud Security Alliance (CSA) STAR-Zertifizierung auf Gold-Ebene.</span><span class="sxs-lookup"><span data-stu-id="3a042-118">Azure, Intune, Power BI, and Microsoft Cloud App Security were awarded Cloud Security Alliance (CSA) STAR Certification at the Gold level.</span></span>
